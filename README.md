📊 AI Job Market Analysis Dataset

This project focuses on analyzing the current trends in the Artificial Intelligence (AI) job market. The dataset has been cleaned and processed using Google Colab to ensure accuracy and usability.

🔍 Project Objectives

Understand demand for AI-related job roles
Identify key skills required in the industry
Analyze salary trends across different roles
Explore job distribution by location and experience level

📁 Dataset Features

Job titles (e.g., Data Scientist, ML Engineer, AI Engineer)
Required skills (Python, Machine Learning, Deep Learning, etc.)
Salary ranges
Company and location details
Experience level

⚙️ Tools Used

Kaggle:- From kaggle the dataset is downloaded.
Python :-Python is used as the coding language.
Pandas & NumPy:-For more addition installed numpy and pandas.
Data visualization libraries
Google Colab:-Where the editing occured with python and kaggle dataset.

🔗Data set link
https://www.kaggle.com/datasets/abhishekjaiswal4896/ai-job-market-trends

📈 Outcome

The project provides meaningful insights into how the AI job market is evolving, helping students and job seekers understand industry expectations and trends.
